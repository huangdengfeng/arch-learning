# Dubbo Admin

Dubbo admin提供较强的运维能力。

实际体验达不到线上运用的标准，后续等新版本出来后，如果有用再展开介绍。

# 安装

1、下载安装包，下载地址open in new window

2、修改bin/config/application.properties中注册地址为nacos.

3、执行bin/start.sh启动后访问 http://localhost:8080 ,默认账号密码root/root
